# **App Name**: Daily Expense Book

## Core Features:

- Expense Tracking: Allows users to quickly log daily expenses with essential details.
- Bill Reminders: Set reminders for recurring bills to ensure timely payments.
- Household Member Management: Add and manage household members to track individual spending contributions.
- Monthly Summaries: Provide a clear overview of monthly spending, broken down by category and member.
- Offline Functionality: Ensure all features are accessible and functional without an internet connection.
- Data Export: Enable users to export their expense data in CSV or PDF format for record-keeping or analysis.
- Google Login: Allow users to log in using their Google account.
- Firebase Backend: Utilize Firebase for data storage and synchronization.
- Ad-Free Subscription: Offer a paid subscription for $0.55/month to remove ads.

## Style Guidelines:

- Primary color: Forest green (#388E3C), symbolizing growth and stability.
- Background color: Light green (#E8F5E9), offering a soft, neutral backdrop.
- Accent color: Teal (#26A69A), providing a contrasting highlight for key actions.
- Body and headline font: 'PT Sans' (sans-serif) to provide a modern, clear aesthetic.
- Simple, intuitive layout optimized for one-hand use, especially on the 'Add Expense' screen.
- Custom icons for expense categories to enhance visual recognition and ease of use.
- Subtle transitions and feedback animations for user interactions to enhance the app's responsiveness.
